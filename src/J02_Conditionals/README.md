# Operators & Conditionals ⚡

In this lesson, Java finally stops being a calculator that only prints stuff and starts making decisions.

## What We Covered

### Operators

The tools Java uses to do things with data.

* Arithmetic (`+`, `-`, `*`, `/`, `%`) → math stuff
* Assignment (`=`, `+=`, `-=`) → updating values
* Increment/Decrement (`++`, `--`) → adding or removing 1
* Comparison (`==`, `!=`, `>`, `<`) → checking things
* Logical (`&&`, `||`, `!`) → combining conditions

### Conditionals

This is where Java starts thinking 🤖

* `if` → one condition
* `if-else` → two possible outcomes
* `if-else-if` → multiple choices
* Nested `if` → conditions inside conditions
* `switch` → cleaner way to handle fixed values

## Things That Hit Different

* `%` (modulus) is lowkey one of the most important operators in DSA.
* `==` checks equality, `=` assigns a value. Mixing them up = pain 💀
* `x++` and `++x` look similar but behave differently.
* Most DSA problems are basically operators + conditions working together.

## TL;DR

Java can now compare values, make decisions, and choose different paths based on conditions.

We're officially past the "Hello World" era 🚀
